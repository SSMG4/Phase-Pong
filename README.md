\# Phase Pong



Phase Pong is a high-velocity, single-input arcade game compiled via a unified C++ codebase for Windows, Linux, Android (NDK), and WebAssembly.



\## Mechanics

\- \*\*Objective:\*\* Match your deflection array's polarity to the incoming particle.

\- \*\*Controls:\*\* Tap the screen (Android/Web) or press Left Click / Spacebar (Desktop) to invert polarity.
